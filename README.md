# 💀 MISS ALIYA - GITHUB REPO HUNTER 💀

## What does it do?
Kisi bhi URL (Render, Vercel, Streamlit, Netlify) ka **GitHub repo ya account** automatically dhundhta hai.

## How to use?
1. URL paste karo
2. "HUNT GITHUB" click karo
3. Agar public repo/account mila toh link dikhega

## Supported platforms
- ✅ Render.com
- ✅ Vercel
- ✅ Streamlit
- ✅ Netlify
- ✅ Heroku
- ✅ GitHub Pages
- ✅ Any website with GitHub link
